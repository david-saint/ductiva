# Product Guide

## Target Users

Developers, designers, and focus-driven professionals who want habit tracking that feels like part of their OS architecture, not a demanding app.

## Primary Goals

- Track daily streaks and visualize long-term progress with macOS-native features.
- Set and manage flexible habit schedules (daily, weekly, monthly).
- Receive smart notifications and reminders tailored to their macOS workflow.

## Key Features

- **Flexible Habit Management:** Create habits with custom schedules and data persistence using SwiftData.
- **System Integration:** System-level notifications and macOS menu bar integration for quick access and reminders.
- **Streak Visualization:** A simple calendar view for each habit to highlight streaks and completion history.
